API Project: Request Header Parser Microservice

By Brandon Roth | Project GitHub

What is this?

This is an API project for freeCodeCamp's Back End Development Certification that returns a JSON object with the user's ip address, language, and operating system.

Usage:

https://brandonr-header-micro.herokuapp.com/api/whoami

Example Output:

{"ipaddress":"your public ip address","language":"en-US","software":"Windows NT 10.0; WOW64"}
